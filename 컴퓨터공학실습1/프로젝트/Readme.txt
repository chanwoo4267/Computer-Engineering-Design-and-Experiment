0-1) 실행 기반은 VS2017 및 VS2019로 준비
0-2) https://openframeworks.cc/download/ 에서 visual studio (2017) version의 openframeworks를 다운받아 압축해제한다.

1) of_v0.11.0_vs2017_release 폴더의 projectGenerator-vs 내부의 projectGenerator을 실행하여 ~\of_v0.11.0_vs2017_release\apps\myApps 경로에 프로젝트를 생성한다.

2) 생성한 프로젝트의 src에 첨부한 src 폴더 내부의 코드를 전부 넣는다. (덮어쓰기)

3) 생성한 프로젝트의 bin\data 폴더에 첨부한 data 폴더 내부의 폰트파일(verdana.ttf) 및 미로를 저장하는 파일(.maz, .txt)를 넣어준다.  .maz는 이전 프로젝트에서 사용한 것과 동일한 규격이어야 하고, .txt는 내부 숫자를 수정하여도 상관없다. 
단, 구성은 반드시 1줄에 스페이스로 나눈 숫자 2개만 들어가야 한다.

4) 생성한 프로젝트의 .sln을 실행하여 프로젝트를 실행한다.
5) 프로젝트->솔루션 대상 변경 에서 windows SDK 버전과 플랫폼 도구 집합을 최신 버전으로 변경한다.
6) 디버그->디버깅시작으로 시작한다.

가능한 기능은 다음과 같다 :

'l' : 미로의 정보가 담겨있는 .maz 파일을 연다.
'g' : 미로의 높이, 폭에 대한 정보가 담겨있는 .txt 파일을 연다.
's' : 시점의 위치를 변경한다. 방향키로 변경할수 있고, 시점의 위치는 검은색으로 표시된다.
'e' : 종점의 위치를 변경한다. 방향키로 변경할수 있고, 종점의 위치는 검은색으로 표시된다.
'd' : dfs 탐색을 수행하고 결과를 출력한다.
'b' : bfs 탐색을 수행하고 결과를 출력한다.
'r' : 금붕어 탐색을 수행하고 결과를 출력한다.
'esc' : 할당된 메모리를 해제하고 프로그램을 종료한다.
